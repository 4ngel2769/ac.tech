## Project Overview

Designed and implemented a centralized smart home system that connects and manages various IoT devices through a single interface, focusing on energy efficiency and automation.

### Key Features

- Device auto-discovery
- Energy usage monitoring
- Automated scheduling
- Mobile app control
- Voice command integration

### Technical Stack

- Raspberry Pi
- Python
- MQTT Protocol
- Node-RED
- InfluxDB for time-series data
